Housing Affordability Dashboard

Analysis of rent burden and income trends in Richland County using ACS data.

Project Overview

This project analyzes housing affordability using census data. It compares income, rent levels, and renter cost burden over time.

Key Findings

  • Nearly half of renters exceed the 30% affordability threshold
  • Income growth does not match housing cost pressure
  • Housing affordability remains consistently strained over time
