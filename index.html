Housing Affordability Dashboard

Analysis of rent burden and income trends using ACS data

Project Overview

Analysis of housing affordability trends in Richland County using census data. Focus on income, rent levels, and renter cost burden over time.

Key Findings

  • High rent burden persists across all periods
  • Income growth does not match housing cost pressure
  • Affordability gap remains stable over time
