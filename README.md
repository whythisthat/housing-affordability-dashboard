\# Housing Affordability Dashboard



\## Overview

Analysis of housing affordability in Richland County using ACS Census data.



\## Tools

\- Python (Pandas)

\- Power BI

\- GitHub Pages



\## Pages

\- Page 1: Housing market trends (income, rent, rent burden)

\- Page 2: Housing affordability stress

\- Page 3: Key insights and findings



\## Key Metrics

\- Rent burden (30%+ income)

\- Median household income

\- Median annual rent



\## Data Sources

U.S. Census Bureau ACS 5-Year Estimates
